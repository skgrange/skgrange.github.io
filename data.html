UK’s National Parks boundaries

These geometries were retrieved with this polygon creation tool which serves OpenStreetMap data.

  • GeoJSON file

London Boroughs boundaries

These geometries are sourced from the London Data Store.

  • GeoJSON file

Simplified boundaries

These geometries are simplified, but high quality boundaries for London’s Boroughs and the City of London. These data are sourced from here which in turn are sourced from sport participation data.

  • GeoJSON file

London low emission zones

The main geometries were drawn manually using this site as the template and the congestion charge zone was sourced from OpenStreetMap.

  • GeoJSON file

Course M25 boundary

This geometry has been created by filtering the road network data and is useful if the “M25 boundary” definition of Greater London is desired.

  • GeoJSON file

Inner London boundaries

There are two definitions for inner London: statutory and statistical. Both of these definitions are found in these geometries. The Simple London Boroughs boundaries were used to form these geometries.

  • GeoJSON file

York’s administrative boundaries

These geometries represent the different levels of York’s administrative boundaries. Data are mostly sourced from the Ordnance Survey boundary line product while the country geometries are from the Global Administrative Areas (GADM) database. The GADM data are not free for commercial use. These geometries do not contain the election boundaries, this is to-do.

  • GeoJSON file

Swiss geometries

Switzerland’s cantons, country boundaries along with crude and approximate language and geographic regions.

  • GeoJSON file
